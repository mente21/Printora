# Logo Organization

## Root Level Logos
- **logo.png** - Main navbar logo (white version) - Primary branding for the site header
- **logo-white.png** - Footer logo (white version for dark backgrounds)

## Brand Logos (in /logos folder)
- **main-logo-white.png** - Primary brand logo (white variant)
- **boa-logo.png** - BOA partner/payment logo
- **cbe-logo.png** - CBE partner/payment logo (PNG)
- **cbe-logo.jpg** - CBE partner/payment logo (JPG)
- **telebirr-logo.png** - Telebirr payment logo (PNG)
- **telebirr-logo.jpg** - Telebirr payment logo (JPG)

## Usage
- **Navbar**: Use `/logo.png` for the main navigation bar
- **Footer**: Use `/logo-white.png` for footer branding
- **Payment Methods**: Reference payment logos from `/logos/` as needed
